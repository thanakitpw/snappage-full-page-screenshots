# SnapPage — Free Full-Page Website Screenshot Tool

## จับภาพหน้าเว็บแบบเต็มหน้า ฟรี ไม่ต้องสมัคร

---

### SnapPage คืออะไร?

เครื่องมือ screenshot เว็บไซต์แบบ full-page ที่ใครก็ใช้ได้ แค่ใส่ URL กดปุ่ม ได้รูปเลย

ไม่ต้องลงโปรแกรม ไม่ต้องสมัครสมาชิก ไม่ต้องจ่ายเงิน

---

### ทำอะไรได้บ้าง?

**1. จับภาพเต็มหน้า (Full Page Screenshot)**
- ได้ทั้งหน้าเว็บจากบนจนล่าง ไม่ใช่แค่ส่วนที่เห็นบนจอ

**2. เลือกขนาดหน้าจอได้**
- Desktop (1920×1080)
- Tablet (768×1024)
- Mobile (375×667)

**3. สแกนหน้าเว็บทั้งหมด (Scan Pages)**
- ใส่ URL หน้าแรก → ระบบจะหาหน้าทั้งหมดในเว็บนั้นให้
- เลือก tick ว่าจะ screenshot หน้าไหนบ้าง
- กดปุ่มเดียว ได้ทุกหน้า

**4. ดาวน์โหลดเป็น ZIP**
- หลายหน้า → รวมเป็นไฟล์ ZIP ให้เลย
- ชื่อโฟลเดอร์เป็นชื่อเว็บ เช่น `example.com-snappage.zip`

**5. ปิด Popup อัตโนมัติ**
- Cookie consent, modal, banner ที่บังหน้า → ระบบปิดให้ก่อนถ่ายภาพ
- ได้รูปสะอาด ไม่มี popup บัง

**6. รองรับ 10 ภาษา**
- English, ไทย, 中文, Español, 日本語, 한국어, Português, Français, Deutsch, Bahasa Indonesia
- เลือกได้จาก dropdown บนหน้าเว็บ

### เทคโนโลยีที่ใช้

| ส่วน | เทคโนโลยี |
|------|-----------|
| Framework | Next.js 15 (App Router) |
| UI | Tailwind CSS + shadcn/ui |
| Screenshot Engine | Playwright + Chromium |
| Hosting | Vercel (Serverless) |
| ภาษา | TypeScript |

---

### วิธีใช้งาน

```
1. เปิดเว็บ SnapPage
2. กรอก URL เว็บที่ต้องการ (เช่น https://example.com)
3. เลือกขนาดจอ + รูปแบบไฟล์ (PNG/JPEG)
4. กด "Capture Screenshot" → ได้รูปเลย
5. หรือกด "Scan Pages" → เลือกหลายหน้า → screenshot ทีเดียว
6. กด Download หรือ Download All (.zip)
```
